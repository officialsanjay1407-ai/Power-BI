# 📊 Dashboard Features

The dashboard provides an interactive overview of business performance through the following visualizations:

### KPI Cards

* **Total Orders**
* **Total Profit**
* **Total Revenue**
* Monthly KPI trend indicators using sparkline visuals

### Interactive Filtering

* Store Location Slicer

  * Airport
  * Commercial
  * Downtown
  * Residential

### Business Performance Analysis

* Total Orders by Product Category
* Total Revenue Trend by Month
* Monthly Profit Trend
* Monthly Revenue Trend
* Monthly Order Trend

### Interactive Capabilities

* Cross-filtering between visuals
* Dynamic filtering using slicers
* Trend analysis using line charts
* Drill Down functionality for time-based analysis
* Interactive business performance monitoring

---

# 💡 Business Insights

This dashboard enables users to:

* Monitor overall business performance through high-level KPIs.
* Compare revenue, profit, and order trends over time.
* Analyze total orders across different product categories.
* Evaluate business performance by store location using interactive filters.
* Identify seasonal trends through monthly revenue and profit analysis.
* Drill down into time-based data for deeper trend analysis.
* Support business decision-making through interactive reporting.
